Jumpstart v2

Version 2 of a quick start template for sites, apps and prototypes - based on the Yeoman Gulp Webapp Generator with added Jade templating.

Download this project as a .zip file Download this project as a tar.gz file

This scaffolding is based on Yeoman - install all the usual dependencies to get that running on your system, then do npm install and bower install to get the site's local dependencies and tooling installed. Once that is done you can gulp serve to spin up the development server with live-reload and gulp to build for distribution.
